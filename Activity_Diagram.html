  • JWT Token validation với .NET Backend
  • Redux state management cho user session

  • DashboardPage.tsx - Tổng quan tài chính

  • Quick actions (Add Income/Expense)
  • Category breakdown panel

  • AddTransactionPage.tsx - Form thêm giao dịch

  • Warning thresholds (70%/90%)

  • Color coding (green/orange/red)
